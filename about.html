About

Lewis Lincoln - UE5 C++ Dev

UE5 C++ C# Visual Scripting PHP HTML CSS Perforce GitHub

I am an experienced Unreal Engine 5 gameplay programmer based in West Yorkshire, focused on C++ systems, Blueprint integration, multiplayer mechanics, UI, and production-relevant gameplay architecture.

Since 2016, I have studied and built a wide range of Unreal projects including inventory systems, combat, AI, multiplayer shooters, GAS, RPG mechanics, dedicated server workflows, destructible environments, and UI systems. My current goal is to convert that experience into a clear, verifiable portfolio.

My experience includes gameplay-focused Unreal Engine systems with emphasis on:

  • C++
  • Blueprint integration
  • Multiplayer architecture
  • Data-driven architecture
  • Scalable implementation

What has always driven my passion for programming? there are many different ways of approaching a problem. I enjoy the decision-making side of programming: choosing the right structure for the problem, the scale of the project, and the people who need to work with the system. Sometimes the simplest approach is correct; other times, a more modular design is needed to support extension, collaboration, and long-term maintainability.

My passion for programming began when I was 16, with scripting and text-based web games. One early influence was Torn City, now known as Torn, and I originally started learning PHP because I wanted to understand and create browser-based game systems. My focus has since moved fully into game development and Unreal Engine, but that early curiosity is what started the path. and since then my passion for games has only grown stronger as I have got older.
