Lewis Lincoln - Portfolio

Unreal Engine Gameplay Programmer

I build gameplay systems in Unreal Engine with a focus on writing code that is practical, understandable, and built for real use inside a game project. For me, programming is not just about getting a feature working once. It is about understanding the purpose of the system, how it will be used, who needs to interact with it, and how it might need to grow later. A good solution depends on context. Sometimes the simplest structure is the right one. Other times, a system needs a more modular approach so designers, artists, and other developers can work with it safely without needing to understand or modify the core implementation. My aim is to build gameplay code that supports the final experience: reliable enough to use, flexible enough to extend, clear enough that other people can understand it, and performant enough for its purpose.
