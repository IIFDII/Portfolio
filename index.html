Lewis Lincoln - Portfolio

Unreal Engine Gameplay Programmer

My experience ensures I build gameplay systems in Unreal Engine with a focus on writing code that is practical, understandable, and built for real use inside a game project. For me, programming is not just about getting a feature working once. It is about understanding the purpose of the system, how it will be used, who needs to interact with it, and how it might need to grow later. A good solution depends on context. Sometimes the simplest structure is the right one; but sometimes a system needs a more modular approach so designers, artists, or other developers can work with it safely. Allowing them to work without needing to touch the core implementation. My aim is to build gameplay code that supports the final experience: reliable enough to use, flexible enough to extend, clear enough that other people can understand it, also performant enough for its purpose.
